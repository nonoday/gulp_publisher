Solid2.0 컴포넌트 사용자 가이드

이 가이드는 Solid2.0 디자인 시스템의 주요 컴포넌트들을 사용하는 방법을 설명합니다.

📦 Infobox 컴포넌트

정보, 성공, 오류, 알림 등의 메시지를 표시하는 컴포넌트입니다.

📁 참고 파일

script/modules/info-box/infobox-component.js

🔧 사용법

<infobox-component type="success" text="고객님은 면책 대상입니다" button-text="자세히 보기" button-link="https://example.com" button-target="_blank" show-icon="true" show-button="true" show-content="true"> <div slot="content"> <p>추가 상세 정보가 여기에 표시됩니다.</p> </div> </infobox-component>

⚙️ 속성 옵션

속성명 타입 필수 기본값 설명
type string 선택 info 인포박스 타입 (success, error, notice, info)
text string 필수 - 메인 텍스트 내용
button-text string 선택 - 버튼에 표시될 텍스트
button-link string 선택 - 버튼 클릭 시 이동할 링크
button-target string 선택 _self 링크 열기 방식 (_self, _blank)
show-icon boolean 선택 true 아이콘 표시 여부
show-button boolean 선택 false 버튼 표시 여부
show-content boolean 선택 false 추가 콘텐츠 슬롯 표시 여부

🧭 Navigation 컴포넌트

페이지 상단의 네비게이션 바를 구성하는 컴포넌트입니다.

📁 참고 파일

script/modules/navigation/navigation-component.js

🔧 사용법

<navigation-component id="mainNavigation"> <!-- Header 슬롯 --> <div slot="header"> <a href="#" class="navigation-header-link"> <img src="../../../../images/icon/icon_chevron_left.svg" alt="뒤로가기" /> </a> <h1 class="navigation-title">타이틀</h1> </div> <!-- Function 슬롯 --> <div slot="function"> <button class="navigation-button" type="button"> <img src="../../../../images/icon/icon_home.svg" alt="홈" /> </button> </div> </navigation-component>

⚙️ 속성 옵션

속성명 타입 필수 기본값 설명
alarm-class string 선택 - 알림 버튼에 적용할 커스텀 클래스

🎯 슬롯 설명

  • header: 왼쪽 영역 (뒤로가기 버튼, 타이틀 등)
  • function: 오른쪽 영역 (기능 버튼들)

➖ Divider 컴포넌트

콘텐츠를 구분하는 구분선 컴포넌트입니다.

📁 참고 파일

script/components/dividers/divider-component.js

🔧 사용법

<!-- 기본 구분선 --> <divider-component type="basic" orientation="horizontal"></divider-component> <!-- 그룹 구분선 (두꺼운) --> <divider-component type="group" orientation="horizontal" color="accent"></divider-component> <!-- 수직 구분선 --> <divider-component type="basic" orientation="vertical" color="brand"></divider-component>

⚙️ 속성 옵션

속성명 타입 필수 기본값 설명
type string 선택 basic 구분선 타입 (basic: 1px, group: 10px)
orientation string 선택 horizontal 구분선 방향 (horizontal, vertical)
color string 선택 - 색상 (accent, secondary, tertiary, brand)
size string 선택 - 크기 (inset: 여백 있음)

🔘 Button 컴포넌트

다양한 스타일과 크기의 버튼 컴포넌트입니다.

📁 참고 파일

script/components/button/solid-button.js

🔧 사용법

<!-- 기본 버튼 --> <solid-button label="기본 버튼"></solid-button> <!-- 색상과 스타일 지정 --> <solid-button label="Primary" color="primary" variant="solid"></solid-button> <!-- 크기와 아이콘 --> <solid-button label="아이콘 버튼" size="l" color="primary"> <span slot="leftIcon">🔍</span> </solid-button> <!-- 캡슐 버튼 --> <solid-button label="캡슐" rounded color="primary"></solid-button>

⚙️ 속성 옵션

속성명 타입 필수 기본값 설명
label string 필수 - 버튼에 표시될 텍스트
color string 선택 primary 색상 (primary, secondary, netural)
variant string 선택 solid 스타일 (solid, outline, ghost, underline)
size string 선택 m 크기 (xl, l, m, s, xs)
rounded boolean 선택 false 캡슐 형태 여부
disabled boolean 선택 false 비활성화 여부
loading boolean 선택 false 로딩 상태 여부

🎯 슬롯 설명

  • leftIcon: 버튼 왼쪽에 표시될 아이콘
  • rightIcon: 버튼 오른쪽에 표시될 아이콘

📝 예시

🏷️ Label 컴포넌트

카테고리나 상태를 표시하는 라벨 컴포넌트입니다.

📁 참고 파일

script/components/label/label-component.js

🔧 사용법

<!-- Solid 타입 --> <label-component type="solid" variant="blue">텍스트</label-component> <!-- Line 타입 --> <label-component type="line" variant="red">텍스트</label-component> <!-- Tint 타입 --> <label-component type="tint" variant="green">텍스트</label-component>

⚙️ 속성 옵션

속성명 타입 필수 기본값 설명
type string 선택 solid 라벨 타입 (solid, line, tint)
variant string 선택 blue 색상 변형 (blue, cyan, green, yellow, orange, red, gray, olive, celery, seaform, indigo, purple, fushia, magenta)

🎨 타입별 설명

  • solid: 배경이 채워진 라벨
  • line: 테두리만 있는 라벨
  • tint: 연한 배경의 라벨

📝 예시

Solid Blue Line Red Tint Green
