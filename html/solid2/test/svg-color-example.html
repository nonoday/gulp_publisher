💡 장점 비교

  • 기존 방법 (filter): 복잡한 수치값, 계산 어려움, 성능 저하
  • 새로운 방법 (currentColor): 피그마 색상값 그대로 사용, 간단함, 성능 좋음

🛠️ 사용법

  1. 피그마에서 아이콘을 SVG로 내보내기
  2. SVG 코드에서 fill="색상값"을 fill="currentColor"로 변경
  3. CSS에서 color 속성으로 색상 제어
