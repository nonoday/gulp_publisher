BottomActionContainer 테마 지원 예시

🎨 테마 시스템

이 예시는 SCSS 변수들을 CSS 변수로 변환하여 다크모드와 라이트모드를 모두 지원합니다. data-theme="dark" 속성을 통해 테마를 전환할 수 있습니다.

기본 사용법
기본적인 BottomActionContainer 사용 예시입니다. 하단에 고정되어 액션 버튼들을 표시합니다. 테마가 변경되면 자동으로 색상이 업데이트됩니다.
메인 콘텐츠 영역
scroll-dim 속성 사용
scroll-dim="true"로 설정하면 상단에 gradient 효과가 적용되어 스크롤 딤 처리가 됩니다. 다크모드에서는 어두운 색상으로 자동 변경됩니다.
스크롤을 위한 더미 콘텐츠
동적 생성 예시
JavaScript를 통해 동적으로 BottomActionContainer를 생성하고 제어하는 예시입니다. 생성된 컨테이너도 현재 테마에 맞게 자동으로 스타일이 적용됩니다.
동적 컨테이너가 여기에 생성됩니다
