동적 툴팁

위치 설정

위치 옵션 설명:
• 기본 위치: 툴팁이 나타날 방향 (상단/하단/좌측/우측)
• 정렬: 선택된 방향에서의 세부 위치 (좌측/중앙/우측 또는 상단/중간/하단)
• 화살표 오프셋: 화살표의 미세한 위치 조정
• 툴팁 오프셋: 툴팁 전체의 위치 조정
• 크기: 툴팁의 넓이와 높이 설정
top-left / top-center / top-right / bottom-left / bottom-center / bottom-right / left-top / left-middle / left-bottom / right-top / right-middle / right-bottom

툴팁 데모 첫번째 버튼만 옵션 변경됨.
