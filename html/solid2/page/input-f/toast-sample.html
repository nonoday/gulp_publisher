Toast 컴포넌트 샘플

기본 사용법

new SolidToast({ message: "기본 토스트 메시지입니다.", align: "center" });

긴 메시지

new SolidToast({ message: "이것은 매우 긴 메시지입니다. 토스트는 자동으로 여러 줄로 표시되며, 높이에 따라 자동으로 표시 시간이 조정됩니다.", align: "left" });

위치 조정

new SolidToast({ message: "위에서 50px 떨어진 위치에 표시됩니다.", align: "center", top: 50 });

HTML 태그 포함

new SolidToast({ message: "첫 번째 줄
두 번째 줄", align: "left" // <br> 태그가 있으면 자동으로 left 정렬 });

로딩 모드

const toast = new SolidToast({ message: "처리 중입니다...", loading: true // 로딩 모드 활성화 (자동으로 숨겨지지 않음) }); // 나중에 숨기기 toast.hide();

색상 옵션

// 검은색 (기본) new SolidToast({ message: "검은색 배경 토스트", color: "black" }); // 흰색 new SolidToast({ message: "흰색 배경 토스트", color: "white" });

모양 옵션

new SolidToast({ message: "커스텀 모양 토스트", shape: "rounded" // 원하는 CSS 클래스명 });

조합 예제

// 로딩 + 흰색 조합 const toast = new SolidToast({ message: "처리 중입니다...", loading: true, color: "white" }); // 메시지 업데이트 toast.updateMessage("처리 완료!"); setTimeout(() => toast.hide(), 2000);
