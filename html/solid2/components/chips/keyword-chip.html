📱 모바일 접근성 개선사항

  • 터치 타겟 크기: 최소 44px (모바일에서는 48px 이상)
  • 반응형 디자인: 화면 크기에 따른 자동 크기 조정
  • 터치 피드백: 터치 시 시각적 피드백 제공
  • 키보드 접근성: Enter, Space, Escape 키 지원
  • ARIA 속성: 스크린 리더 지원
  • 선택 상태: selected 속성으로 선택/해제 가능
  • 삭제 기능: removable 속성으로 X 버튼 표시

🗑️ 삭제 가능한 키워드칩

아래 칩들은 removable 속성이 있어 X 버튼이 표시됩니다. X 버튼을 클릭하거나 Delete/Backspace 키를 누르면 칩이 삭제됩니다.

기본 사용법

Type 옵션

type="tonal" Tonal 스타일
type="line" Line 스타일

Color 옵션

color="gray" Gray 색상
color="blue" Blue 색상
color="red" Red 색상

Type과 Color 조합

type="tonal" color="blue" Blue Tonal
type="line" color="red" Red Line

상태 관리

selected 선택된 칩
disabled 비활성화된 칩
